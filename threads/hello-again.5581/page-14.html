I don't know what you're doing in trying to create a new thread, but once in the index of any particular section, there is a POST button...to the left of PREVIOUS,INDEX,NEXT,EXPAND and COLLAPSE, and has a little page icon in it.

IF you have the preview box clicked, then a preview will appear first, and you will have to click on the continue button in the "preview" window for the new thread to be published.

The image posting is a bit tougher. Right click on whatever image you want to post here, and select and click on properties from the fly-out that should appear. The properties window will display the images URL address, which you highlight and copy. Then coming back to TTI, in your draft ( not the preview ) window, click on the IMAGE button in the Instant UBB Code list of options. This will bring up a http:// box to paste what you copied. You have to make sure that http:// doesn't accidently go in twice, which happens sometimes.


Then just click on OK, and it should appear in your draft window, however, you won't see the image until you either preview the post, or publish it.

AND if you go back into "edit" the post after you already published it onto the forum, it messes up the image code line and you have to do the IMAGE posting process again. I usually will click on my posted image here, re-copy the properties again, then edit the post, and re-do the image without having to go anywhere on the internet to retrieve the property info again.
