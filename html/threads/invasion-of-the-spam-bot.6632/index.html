I deleted a bunch of spam this morning. I proceed as follows:

- I click the IP link underneath the post and use that information to ban both the IP address (the entire range) and the username in the Admin Panel.
- I then permanently delete the message.

By having one window open on the forum and another on the Admin Panel, it goes pretty quickly.

I've also "moved the cheese". Which is to say, the spambots are currently programmed to proceed in a certain fashion. So I've changed registration, disabling ReCAPTCHA and enabled Q&A CAPTCHA (questions like "Freshly fallen snow is what color?") They'll eventually work around that, at which point I'll turn on ReCAPTCHA again.
